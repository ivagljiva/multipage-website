Research

I conduct research in the fields of computational biology and bioinformatics. My current focus is on comparative genomics of prokaryotes. I also write Perl and Python scripts for parsing and analysis of my data. In the future, I want to write accessible, open-source genomics software to help me and other researchers conduct experiments and manage data more effectively.

Automated Coloring of Phylogeny Figures from MrBayes

While studying the MrBayes application for the Student Cluster Competition, I developed a Perl script that automatically adds colors to the consensus tree (.tre) file based on the start of the species name. When the file is viewed in a software such as FigTree, you see a tree such as the one above, in which each clade is a different color. This allows for easier analysis of evolutionary relationships. It proved useful in the competition for the identification of mislabeled species, which was one of our MrBayes challenges.
